# 🎬 Netflix GPT

A smart movie recommendation application built using **Create React App**, **TailwindCSS**, and **TMDB APIs**.  
This project includes authentication, personalized browsing, and an AI-powered search experience.

---

## 🚀 Tech Stack
- **React (CRA)**
- **TailwindCSS**
- **Firebase Authentication**
- **TMDB API**

---

## ⭐ Features

### 🔐 Authentication
- Sign In / Sign Up form  
- Validation & error handling  
- Redirects to Browse page after login  

---

### 🎥 Browse Page (Only after Authentication)
- Responsive **Header**
- **Main Movie section**
  - Trailer playing in the background  
  - Movie Title & Description overlay  
- **Movie Suggestions**
  - Multiple category-wise movie lists  
  - Smooth horizontal carousels (MovieList × N)

---

### 🤖 Netflix GPT (AI-powered Search)
- Smart Search Bar  
- GPT-based movie suggestions  
- Real-time results & smooth UI
